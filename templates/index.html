Random Forrest Model

Author: Daniel Lagos

Random Forrest that will classify dry beans based on 9 inputs. 7 possible classificaitons

All inputs are numbers, they can have decimal's

Data set from UCI Machine Learning Repository: Dry Bean Dataset. 7 different types of dry beans were used in this classification problem. A computer vision system captured images out of which 16 different features were quantified. A total of 13,611 observations were used in this research. The design of the original research seemed to combine computer vision and machine learning for this classification problem, this model attempts to recreate the machine learning portion of the original research. Original research did not consider a random forest, therefore, this assignment will focus on a random forest in the classification problem.

Problem statement: Can any of the 16 features found in the Dry Bean Data Set be used to identify (classify) the 7 different registered varieties of dry beans?

After a correlation analysis the 16 features were reduced to 9 features. Model produced about 92% accuracy score. F-1 scores for each class were between 89% to 100%.
